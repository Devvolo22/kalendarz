# Kalendarz urlopów 2026 — look & feel jak na pokazanej stronie

Zaktualizowałem UI tak, żeby był **wizualnie zbliżony** do pokazanego przykładu:
- jasnoszare tło,
- duża, mocna typografia hero,
- górny pasek + nawigacja,
- zielony przycisk CTA,
- czarny przycisk drugorzędny,
- proste ikony i czyste, jasne panele.

## Widoki kalendarza
- Widok miesięczny.
- Widok roczny (12 miesięcy).
- W obu widokach widoczne są: święta, urlopy i niestandardowe dni wolne.

## Uruchomienie
```bash
python3 -m http.server 8000
```
Następnie otwórz `http://localhost:8000`.
